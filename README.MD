# 🚀 Portfolio - Hyan Bispo Felix

Este é o meu portfólio pessoal e profissional, desenvolvido para centralizar meus projetos de Engenharia de Software, automações com IA e experiências em Service Desk.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído focado em **performance** e **design responsivo**, utilizando:

* **HTML5 & JavaScript (ES6+)**
* **Tailwind CSS:** Para uma estilização moderna, rápida e utilitária.
* **AOS (Animate On Scroll):** Para animações fluidas durante a navegação.
* **Font Awesome:** Para iconografia técnica e social.
* **Google Fonts (Inter):** Foco em legibilidade e interface limpa (Clean UI).

---

## 🌟 Destaques do Projeto

* **Design Glassmorphism:** Interface moderna com efeitos de desfoque e transparência.
* **Totalmente Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
* **Otimização de SEO:** Tags semânticas para melhor ranqueamento e acessibilidade.
* **Call to Action (CTA):** Botão de "Copiar E-mail" interativo para facilitar o contato.
